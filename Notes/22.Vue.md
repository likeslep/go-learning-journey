# Vue

## 创建项目
确保安装了Node.js，并且当前目录正好是打算创建项目的目录。在命令行中运行以下命令：
```
npm create vue@latest
```
这个指令将会安装并执行create-vue，它是Vue官方的项目脚手架工具。会看到一些可选功能的提示：

```

✔ Add TypeScript? … No / Yes
✔ Add JSX Support? … No / Yes
✔ Add Vue Router for Single Page Application development? … No / Yes

✔ Add Vitest for Unit testing? … No / Yes
✔ Add an End-to-End Testing Solution? … No / Cypress / Nightwatch / Playwright
✔ Add ESLint for code quality? … No / Yes
✔ Add Prettier for code formatting? … No / Yes
✔ Add Vue DevTools 7 extension for debugging? (experimental) … No / Yes

Scaffolding project in ./<your-project-name>...

```
如果不确定开启哪个功能，可以直接按回车键选择No。在项目创建后，通过在项目目录下执行：`npm install`来安装依赖。

总结：
- `Vite`项目中，index.html是项目的入口文件，在项目最外层。
- 加载index.html后，Vite解析`<script type="model" src="xxx">`指向的`JavaScript`。
- `vue3`中是通过`createApp`函数创建一个应用实例的。


## vue3中兼容vue2的语法
```vue
<template>
 <div class="person">
    <h2>姓名：{{ name }}</h2>
    <h2>年龄：{{ age }}</h2>
    <button @click="changeName">修改名字</button>
    <button @click="changeAge">修改年龄</button>
    <button @click="showTel">查看联系方式</button>
 </div>
</template>

<script lang="ts">
    export default {
        name:'Person',
        data() {
          return {
            name:'张三',
            age:18,
            tel:'12388888888'
          }   
        },

        methods:{
            changeName(){
                this.name='zhang-san'
            },

            changeAge(){
                this.age += 1
            },

            showTel(){
               alert(this.tel) 
            }
        }
    }
</script>

<style scoped>
    .person {
        background-color: skyblue;
        box-shadow: 0 0 10px;
        border-radius: 10px;
        padding: 20px;
    }
</style>
```


## vue3 setup()

```vue
<template>
  <div class="person">
    <h2>姓名：{{ name }}</h2>
    <h2>年龄：{{ age }}</h2>
    <button @click="changeName">修改名字</button>
    <button @click="changeAge">修改年龄</button>
    <button @click="showTel">查看联系方式</button>
  </div>
</template>

<script lang="ts">
export default {
  name: 'Person',
  setup(){
    let name = '张三'
    let age = 18
    let tel = '12388888888'

    function changeName(){
        name = 'zhang-san'
    }

    function changeAge(){
        age += 1
    }

    function showTel(){
        alert(tel)
    }
    return {name,age,changeName,changeAge,showTel}
  }
}
</script>

<style scoped>
.person {
  background-color: skyblue;
  box-shadow: 0 0 10px;
  border-radius: 10px;
  padding: 20px;
}
</style>
```

## setup()语法糖
```vue
<template>
  <div class="person">
    <h2>姓名：{{ name }}</h2>
    <h2>年龄：{{ age }}</h2>
    <button @click="changeName">修改名字</button>
    <button @click="changeAge">修改年龄</button>
    <button @click="showTel">查看联系方式</button>
  </div>
</template>

<script lang="ts" setup name="Person">
    let name = '张三'
    let age = 18
    let tel = '12388888888'

    function changeName() {
        name = 'zhang-san'
    }

    function changeAge() {
        age += 1
    }

    function showTel() {
        alert(tel)
    }
</script>

<style scoped>
</style>
```
不需要写setup()函数并且返回里边的数据了。这里只写了一个script，是因为我装了`vite-plugin-vue-setup-extend`插件，这样我可以在script标签中直接定义组件名字，就不需要再写一个script来定义组件名字了。


## ref
上边定义的数据都不是响应式的，下边才是，要想对数据进行更改，要用到ref
```vue
<script lang="ts" setup name="Person">
    import {ref} from 'vue'

    let name = ref('张三')
    let age = ref(18)
    let tel = '12388888888'

    function changeName() {
        name.value = 'zhang-san'
    }

    function changeAge() {
        age.value += 1
    }

    function showTel() {
        alert(tel)
    }
</script>
```
