# 2026年2月12日

- **Go预定义日期和时间戳格式常量**
  - 年 ：`2026`；`06`
  - 月 ：`Jan`；`January`；`01`；`1`
  - 日 ：`02`；`2`；`_2`
  - 星期 ：`Mon`；`Monday`
  - 小时 ：`15`；`3`；`03`
  - 分钟 ：`04`；`4`
  - 秒 ：`05`；`5`
  - 上午/下午标记(AM/PM) ：`PM`
  - yearDay（一年中的某一天）：`002`；`_ _2`

- **比较时间前后**
```go
t1.After(t2)  // 判断t1是否在t2之后
t1.Befor(t2)  // 判断t1是否在t2之前
```

- **将字符串解析为特定的值**
`time.Parse(layout, date)` layout根据特定的时间戳的值来根据date编写

- **将时间类型转化为特定的字符串**
`t.Format(layout)` layout根据特定的事件戳的值来编写

- **获取时间的各个部分**
  - `t.Weekday()`
  - `t.Year()`
  - `t.Month()`
  - `t.Day()`
  - `t.Hour()`
  - `t.Minute()`
