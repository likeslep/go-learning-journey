题目：给出以下程序的输出结果
```go
package main

const s = "Go101.org"
// len(s) == 9
// 1 << 9 == 512
// 512 / 128 == 4

var a byte = 1 << len(s) / 128
var b byte = 1 << len(s[:]) / 128

func main() {
  println(a, b)
}
```
答案：4 0 

关键点在于 **常量表达式** 和 **变量表达式** 在Go中的不同计算规则。

1. 常量表达式：`1 << len(s) / 128`
   - `s`是常量字符串，`len(s)`在编译时确定为9
   - 由于整个表达式操作数都是常量，是常量表达式，在编译时直接计算出结果，所以a的值为4。

2. 变量表达式：`1 << len(s[:]) / 128`
